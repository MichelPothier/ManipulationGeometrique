# ManipulationGeometrique
Outils pour décrire, visualiser, manipuler et modifier les éléments à partir de géométries de travail. / Tools to describe, visualize, manipulate and modify elements from working geometries. (ArcGis Pro 2.0 / ArcGis 10.5 / ArcMap / Add-in / Vb.Net)

OngletGeometrieTravail (ArcGis Pro 2.0 / Add-in / Vb.Net)
-Onglet utilisée dans dans ArcGis Pro 2.0 pour décrire, visualiser, manipuler et modifier les éléments à partir de géométries de travail.
 Pour plus de détails, voir le fichier de documentation "OngletGeometrieTravail.doc" présent dans ce dépôt.
-Tab used in in ArcGis Pro 2.0 to describe, view, manipulate, and modify elements from working geometries.
 For more details, see the documentation file "OngletGeometrieTravail.doc" present in this repository.

BarreGeometrieTravail (ArcGis 10.5 / ArcMap / Add-in / Vb.Net)
-Barre de menu utilisée dans dans ArcMap pour décrire, visualiser, manipuler et modifier les éléments à partir de géométries de travail. 
 Pour plus de détails, voir le fichier de documentation "BarreGeometrieTravail.doc" présent dans ce dépôt.
-Menu bar used in in ArcMap to describe, view, manipulate, and modify elements from working geometries.
 For more details, see the documentation file "BarreGeometrieTravail.doc" present in this repository.
 
BarreUniteTravail (ArcGis 10.5 / ArcMap / Add-in / Vb.Net)
-Barre de menu utilisée dans dans ArcMap pour décrire, visualiser, manipuler et modifier les éléments à partir de géométries calculées selon le Système Nationale de Référence Cartographique du Canada (SNRC). 
 Pour plus de détails, voir le fichier de documentation "BarreUniteTravail.doc" présent dans ce dépôt.
-Menu bar used in ArcMap to describe, visualize, manipulate and modify elements from geometries calculated according to the National Topographic System of Canada of Canada (NTS)
 For more details, see the documentation file "BarreUniteTravail.doc" present in this repository.
